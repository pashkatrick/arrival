видимо нужно менять значение пинов в зависимости от кейса и отслеживать изменение сигналов

## Cases

```
Переключение передачи в Drive

- достаточное напряжение на контактах батареи (BatteryState == Ready)
- педаль тормоза нажата (BrakePedalState == Pressed)
- педаль газа в положении 1 (AccPedalPos == 0%)
- напряжение на пинах Gear Shifter == напряжение требуемое для переключения в Drive

Ожидаемый результат: Сигнал GearPosition == Drive
```


```
Батарея недосутпна

- отсутствие напряжения, повышенное напряжение, короткое замыкание (BatteryState == Error)

Ожидаемый результат: нельзя переключить передачу, значение сигнала GearPosition == Neutral, пины == 0, сигналы == Error
```

```
Низкое напряжение в батарее

- отсутствие напряжения, повышенное напряжение, короткое замыкание (BatteryState == NotReady)

Ожидаемый результат: нельзя переключить передачу, значение сигнала GearPosition == Neutral
```


```
Переключение передачи на ходу

- достаточное напряжение на контактах батареи (BatteryState == Ready)
- педаль тормоза нажата (BrakePedalState == Released)
- педаль газа в положении 2 (AccPedalPos == 30%)
- напряжение на пинах Gear Shifter == напряжение требуемое для переключения в Park

Ожидаемый результат: Переключение передпчи невозможно, GearPosition == Neutral
```

```
Нажатие на педаль тормоза с зажатым газом

- педаль нажата (BrakePedalState == Pressed)
- педаль нажата (AccPedalPos == 30%)

Ожидаемый результат: значение сигнала ReqTorque == 0Nm
```

```
Нажатие на педаль тормоза с отпущенным газом

- педаль нажата (BrakePedalState == Pressed)
- педаль нажата (AccPedalPos == 0% or AccPedalPos == Error)

Ожидаемый результат: значение сигнала ReqTorque == 0Nm
```

```
Движение с зажатым тормозом

- педаль нажата (BrakePedalState == Pressed)
- педаль нажата (AccPedalPos == 30% or AccPedalPos == Error)
- включена D или R на коробке (GearPosition == Drive or Reverse)

Ожидаемый результат: значение сигнала ReqTorque == 0Nm
```
